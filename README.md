### How to use
1. Make sure you have [**CocoaPods**](http://cocoapods.org/) installed
2. `pod install`
3. open *FakeZhihuDaily.xcworkspace*, build & run
4. If you have any questions, feel free to contact me [Mingyu Cui](mailto:malvin.cui@gmail.com)
